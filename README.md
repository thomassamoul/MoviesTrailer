# MoviesTrailer

Showing list of popular movies and series on themoviedb.com . Showing movie-series details when list item clicked

## Features

- [x] Java
- [x] MVVM Architecture
- [x] RecyclerView
- [x] Retrofit for Networking
- [x] RX for Threading
- [x] Room persistence for caching
